Youtuber

A partir de 13/02/2022 comecei a lançar vídeos na plataforma Youtube com o objetivo de repassar os conhecimentos adquiridos ao longo dos anos como programador.

A primeira aula do canal foi sobre o assunto de Flexbox, uma técnica CSS para tornar uma página web responsiva com menos esforço.

Para o futuro, estão planejadas aulas nas seguintes vertentes: ciência de dados, programação web e automação de tarefas.
